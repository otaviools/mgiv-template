## 📌 Sobre o Projeto
A **MGIV** é uma empresa acreditada pelo Inmetro/Senatran especializada em regularização e inspeção veicular. O desafio foi reformular sua presença digital, substituindo um sistema obsoleto por uma solução moderna, rápida e otimizada para conversão.

---

## 🚀 Metodologia STAR

### **S (Situation) - Situação**
A empresa possuía uma identidade visual defasada e um site com baixa performance mobile, o que dificultava o ranqueamento orgânico e a experiência de motoristas/transportadores que acessam o site em trânsito (rodovias).

### **T (Task) - Tarefa**
Reestruturar toda a identidade visual e desenvolver uma Landing Page "Vanilla" (sem frameworks pesados), priorizando a semântica de código, velocidade de carregamento e um backend seguro para captação de leads.

### **A (Action) - Ação**
- **UI/UX Design:** Prototipação completa no **Figma**, criando uma interface limpa e intuitiva.
- **Desenvolvimento Nativo:** Uso de **HTML5 Semântico**, **CSS3 (Modern Layouts)** e **JavaScript Vanilla**.
- **Backend Robusto:** Implementação de script em **PHP** para processamento de formulários.
- **Engenharia de Performance:** Otimização de ativos com **WebP**, minificação de código e carregamento crítico.
- **Segurança:** Validação estrita via **Regex** e verificação de **DNS (MX)** no backend para filtragem de SPAM.

### **R (Result) - Resultado**
- **Score 100%** no Google PageSpeed Insights.
- Interface 100% responsiva e acessível.
- Backend blindado contra entradas falsas, garantindo integridade nos leads recebidos pela empresa.

## 📈 Performance (Google PageSpeed)
O projeto foi otimizado para atingir a nota máxima em performance, garantindo que usuários com conexões instáveis (3G/4G) em rodovias tenham acesso imediato às informações.

---

## 🛡️ Segurança de Dados
O formulário de contato conta com uma camada de segurança que realiza a checagem do domínio do e-mail informado (verificação de registro MX), impedindo que bots e endereços inexistentes sobrecarreguem o servidor SMTP da cliente.
